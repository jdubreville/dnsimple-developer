DNSimple API

Template API

Templates provide a way to group common records together and apply them en masse to a domain.

  • List templates
  • Get a template
  • Create a template
  • Delete a template

List templates

GET /templates

Example

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      https://dnsimple.com/templates

Response

[
  {
    "dns_template": {
      "name": "Example",
      "created_at": "2010-09-30T20:01:55Z",
      "updated_at": "2010-09-30T20:01:55Z",
      "id": 31,
      "user_id": 1,
      "short_name": "example",
      "description": "An example of a template."
    }
  }
]

Get a template

GET /templates/:id

Example

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      https://dnsimple.com/templates/:id

Response

{
  "dns_template": {
    "name": "Example",
    "created_at": "2010-09-30T20:01:55Z",
    "updated_at": "2010-09-30T20:01:55Z",
    "id": 31,
    "user_id": 1,
    "short_name": "example",
    "description": "An example of a template."
  }
}

Create a template

POST /templates

Example

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X POST \
      -d '<json>' \
      https://dnsimple.com/templates

Input

The following fields are required:

  • dns_template[name]
  • dns_template[short_name]

The following fields are optional:

  • dns_template[description]
{
  "dns_template": {
    "name": "My Template",
    "short_name": "my_template" 
  }
}
